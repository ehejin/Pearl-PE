## Run Experiments

To run PEARL on the REDDIT-BINARY and REDDIT-MULTI datasets, open the PEARL-REDDIT directory and run
```
python -m train.zinc --config reddit-m.yaml
python -m train.zinc --config reddit-b.yaml
```
